# OA办公管理系统 java484

## 项目概述


## 技术栈

- 后端：Spring + SpringMvc + Mybatis
- 前端：Vue.js
- 数据库：MySQL
- 项目管理：Maven

## 功能模块

- 用户管理
- 部门管理
- 文档管理
- 工作流程
- 日程安排
- 报告生成

## 系统角色

- 系统管理员
- 普通用户
- 部门经理

## 运行环境

- 服务器：Tomcat 8.x 或更高版本
- 数据库：MySQL 5.x 或更高版本
- 操作系统：Windows/Linux均可
- 浏览器：支持现代浏览器（如Chrome、Firefox等）

## 部署步骤

1. 准备所需环境：配置好Java和MySQL环境
2. 导入MySQL数据库：使用提供的数据库脚本进行初始化
3. 部署后端服务：将项目war包部署到Tomcat服务器
4. 部署前端页面：将Vue.js项目构建后部署到Web服务器相应目录
5. 访问系统：通过浏览器访问部署好的系统地址

## 目录结构

```
oa_system/
├── src/            # 源代码目录
│   ├── main/
│   │   ├── java/     # 后端代码
│   │   ├── resources/ # 配置文件
│   │   └── webapp/     # 前端代码
│   └── test/         # 测试代码
├── pom.xml          # Maven项目文件
└── database.sql     # 数据库脚本
```

## 核心亮点

- 界面简洁，操作直观，提升办公效率
- 前后端分离设计，易于维护和扩展
- 基于B/S结构，支持多平台访问
- 严格遵循MVC架构，代码结构清晰
